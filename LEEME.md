# 0_Album_Financiero_original

Este repositorio presenta la versión original del documento Album Financiero, el documento personal que utilicé para registrar y comprender mis finanzas durante más de diez años. 

El archivo de Excel, anonimizado para proteger información sensible, concentra todo el proceso de captura y análisis de datos en un solo libro dividido en hojas como C. Diaria, Presupuesto, Inversiones, Deudas y Viajes, que abarcan transacciones, presupuestos, inversiones, deudas y gastos de viaje. 

Aunque no aplica reglas de normalización ni procesos automáticos, el documento cumplió su función organizativa y sentó las bases para el proyecto de Power BI publicado en otro repositorio. 

Este repositorio incluye el archivo anonimizado y un video explicativo como referencia.
